\# Importador RCV SII Chile – Odoo 18 Enterprise



Módulo base para importar el Registro de Compras y Ventas (RCV)

desde el SII en Chile.



\## Requisitos

\- Odoo 18 Enterprise

\- Odoo.sh

\- Módulos:

&nbsp; - account

&nbsp; - l10n\_cl

&nbsp; - l10n\_cl\_edi



\## Estado

\- Estructura base

\- Modelos definidos

\- Wizard operativo

\- Sin conexión SII (fase siguiente)



Instalación vía GitHub compatible con Odoo.sh.
